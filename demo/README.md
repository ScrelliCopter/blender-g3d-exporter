## Run with CLI
1. `git clone https://github.com/haz00/blender-g3d-exporter.git`
2. `cd blender-g3d-exporter/demo`
3. `./gradlew run`

## Run with IntelliJ IDEA
1. `git clone https://github.com/haz00/blender-g3d-exporter.git`
2. IDEA - File - Open - demo folder 
3. Run DesktopLauncher as usual
